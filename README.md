# On-Device Personalized Recommendation System

This example demonstrates the core concept of "on-device training" for personalized applications. It simulates a recommendation system where a user's preferences are updated locally on the "device" (the script) based on their interactions (item ratings). This approach enhances user privacy by keeping data on the device and allows for immediate personalization without relying on cloud services.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
